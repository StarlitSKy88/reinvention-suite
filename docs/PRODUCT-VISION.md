# 产品定位：从 0 到 100 万 AI 教练

> **核心洞察**：现有产品（35+ 失业再就业）太"自嗨"，目标人群窄、To-G 模式慢、需要太多前置工作
> **新定位**：面向"一无所有的人"的 AI 赚钱教练，**收费测试**，从付费意愿开始

## 🎯 新产品一句话

> **"一无所有的你，如何从 0 挣到 100 万"**
> AI 通过 5-8 个互动提问，诊断你的现状、资源、优势，输出"从 0 到 100 万"的具体路径

## 🎯 目标用户（主次分明）

### 主：普通人想赚钱
- 应届生 / 大学生（无资源、有时间）
- 35+ 失业（已经在做）
- 宝妈 / 全职妈妈（想副业）
- 小镇青年（无背景）
- 上班族想副业
- 创业者初期（无资金、无经验）

### 次：35+ 失业（保留现有项目）
- 作为"失业后如何从 0 开始"的子集
- 现有 reinvention-suite 提供垂直深度（简历、岗位、项目）

## 💎 核心价值主张

> **"5 分钟 AI 诊断 = 一份'你的 100 万路径'地图"**

不是：
- ❌ 写简历让 AI 改（功能层面）
- ❌ 推荐岗位（执行层面）
- ❌ 提供课程（教育层面）

而是：
- ✅ 像咨询师一样**追问**
- ✅ 像教练一样**给方案**
- ✅ 像军师一样**看你的牌**

## 🤖 AI 诊断流程

```
Step 1: 用户支付 ¥99（收费测试）
   ↓
Step 2: AI 提问（5-8 个深度问题）
   "你现在的处境是什么？"
   "你过去最挣到钱的一次是怎么做的？"
   "你身边有什么资源/人脉/技能？"
   "你能投入多少时间？"
   "你能承受多少风险？"
   "你最擅长什么（即使没人付钱）？"
   ↓
Step 3: AI 综合分析
   - 现状定位（无资源 / 有时间 / 有技能）
   - 起点评估（现有最低可变现资源）
   - 路径推荐（3 条"0 到 100 万"路径）
   ↓
Step 4: 输出 3000 字"你的 100 万路径"PDF
   - 你的起点评估
   - 三条可行路径（每条 30 天 / 90 天 / 1 年目标）
   - 每条路径的"最小第一步"
   - 风险与避坑
   - 案例对标
   ↓
Step 5: 后续 30 天邮件陪伴（可选加价）
```

## 💰 变现模式

### 核心：¥99/次（低门槛跑量）

| 项 | 价格 | 转化率 | 备注 |
|---|---|---|---|
| **AI 诊断 + 方案** | ¥99 | 5% | 核心 |
| 30 天邮件陪伴 | +¥199 | 20% | 升级 |
| 1v1 教练 60 分钟 | +¥999 | 5% | 高净值 |

## 🤖 AI 诊断 Prompt 框架

### 提问阶段（5-8 个问题）

```
Q1: "你现在的处境是什么？（失业 / 上班族 / 学生 / 创业 / 其他）"
Q2: "你过去有没有挣到过钱？最多一次是多少？是怎么做到的？"
Q3: "你身边有什么资源？（人脉 / 技能 / 设备 / 时间 / 资金）"
Q4: "你每天能投入多少时间？（1h / 3h / 全职）"
Q5: "你最能承受多少风险？（不能亏损 / 能亏损 1 万 / 能更多）"
Q6: "你最擅长什么？即使没人付钱你也愿意做？（写作 / 销售 / 编程 / 设计 ...）"
Q7: "你身边有没有成功的人？你最想成为谁？"
Q8: "你希望在多久内挣到 100 万？（1 年 / 3 年 / 5 年 / 不设期限）"
```

### 综合分析 Prompt

```
你是中国顶级商业咨询师。请基于用户 8 个回答，给出：

1. 起点评估（100 字以内）
2. 三条可行路径：
   - 路径 A："低成本快速启动"（30 天可见收入）
   - 路径 B："中等投入稳步增长"（3-6 月达 10 万）
   - 路径 C："长期复利型"（1-3 年达 100 万）
3. 每条路径的：
   - 核心策略
   - 90 天行动清单
   - "最小第一步"（今天就能做）
   - 真实案例对标
4. 共同的 3 个陷阱
5. 最后给一句："如果你只听我一句话..."

要求：基于用户实际回答，不要给通用建议。要具体、要可执行、不要鸡汤。
```

## 💡 关键成功因素

1. **AI 诊断质量**（90% 的价值）→ 写好 prompt
2. **付费意愿验证**（先有需求再做产品）→ ¥99 是合理价格
3. **真实案例**（>一切营销）→ 第一批用户必须有结果
4. **口碑传播**（>一切投放）→ 让用户帮我们传播
