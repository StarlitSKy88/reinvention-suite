# 政府再就业效果报告模板 v1.0（Task #14）

> **用途**：试点期结束或年度合同时给政府领导的"成绩单"
> **设计**：黑底白字红字点缀（与产品风格一致）
> **核心目标**：让领导觉得"钱花得值，明年继续"

---

## 📑 报告结构

```
1. 封面（领导专属）
2. 执行摘要（1 页）
3. 试点概述（2 页）
4. 核心成果（5 大模块，6 页）
5. 标杆案例（3 个深度故事，3 页）
6. 政府投入产出分析（2 页）
7. 客户反馈（NPS + 声音，2 页）
8. 媒体宣传机会（1 页）
9. 下一阶段建议（2 页）
10. 联系方式（1 页）
```

## 📝 附录：数据采集说明

### 数据来源
- 平台用户行为日志
- 用户主动申报的成功案例
- 第三方核验（社保、个税 APP 授权）

### 脱敏规则
- 真实姓名替换为化名
- 联系方式部分隐藏
- 公司名称按行业泛化（如"某互联网公司"）
- 薪资数据保留量级，具体数字四舍五入

### 数据准确性
- 平台行为数据：100% 准确
- 成功率：基于用户主动申报 + 30 天在职核验
- 标杆案例：用户授权公开
